# I3WM Configuration

This directory contains the i3 config files for my system

## Requirements

Ensure you have the following installed on your system

#### Alacritty

```
pacman -S alacritty
```

#### Firefox Browser

```
pacman -S firefox
```

#### Better Lock Screen

```
pacman -S betterlockscreen
```

#### Nitrogen
For backgrounds
```
pacman -S nitrogen
```

#### Polybar
Status bar for i3
```
pacman -S polybar
```
